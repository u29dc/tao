# tao-macos

Native macOS Swift application shell for `tao`.

This app must remain a UI adapter over Rust SDK services and must not contain domain logic.

## Planned Targets

- TaoApp
- TaoFeatureNavigation
- TaoFeatureNote
- TaoFeatureProperties
- TaoFeatureBases
- TaoBridgeClient
- TaoDesignSystem
- TaoTestingSupport

## Status

Architecture boundary scaffold complete. Xcode project bootstrap is tracked by `APP-001`.
